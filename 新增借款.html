交互说明及约束规则：


1. 新增借款信息（发标）时，还款方式增加“按月付息+分次还本”、“按月付息+等额本金”、“按月付息+等额本息”等三项；（ps. 在“标产品管理”模块修改产品信息时也需在“还款方式”行增加相应选项）


2. 选择“按月付息+分次还本”还款方式时，需指定还本次数（必填），还本次数不得大于借款期限，否则在还款方式行尾红字提示“还本次数不得大于借款期限！”


3. 系统根据操作人员录入的还本次数生成本金偿还方案列表，除最后一期外，其余各期均可手动指定偿还月份及偿还本金金额，偿还月份（第x个月）需满足 x < n，n即借款期限，否则在该行尾红字提示：指定偿还月份必须小于借款期限！；


4. 最后一期偿还本金金额 = 借款金额 - 其余各期还本金额之和，由系统自动实时计算，且最后一期偿还本金金额不能小于等于0，否则在列表右侧红字提示：最后一个月偿还本金金额不得小于等于0！


5. 选择“按月付息+等额本金”或“按月付息+等额本息”还款方式时，需指定前x个月按月付息，同时根据借款期限n自动实时计算后(n-x)个月等额本金或等额本息，并且需满足 x <= n -2 ，否则在行尾红字提示：先付息月数不得大于(n-2)！；


6. 分次还本次数、偿还本金、先付息月数均为正整数；
